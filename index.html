Accueil Équipe

Graph Labs est une organisation indépendante qui vous propose diverses prestations autour de Neo4j, la base de données graphe native :

  • ateliers Cypher d'une demi-journée
  • formations déclinables en 1, 2 ou 3 journées selon vos besoins
  • sessions sur mesure
